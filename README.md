#  Incident Reporting Tool

A Python-based tool that simulates an internal operations monitoring system.
The program collects incident data, analyzes severity and downtime metrics,
and generates a structured operational status report.

## Features
- Collects incident data via user input
- Tracks unresolved and high-severity incidents
- Calculates average downtime metrics
- Generates a clean operational status report
- Uses clean functional design (no global state)

## Technologies
- Python
- Git

## Purpose
This project was created to practice Python fundamentals, data modeling,
and clean program structure for internal tooling and automation use cases.
